Qual a diferença de var, let e const?

var,let,const

A inclusão no EC6 do const e let foi para suprir problemas causados pelo uso do 'var', o var sofre o hoisting, de escobo global ou estrito dependendo de onde é declarado.

var,let,const

A variavel var pode ser redeclarada, sem gerar erro e podemos alterar seu valor

estas características tornam o var muito conveniente a erros

var,let,const

Assim veio o 'let' que usa das propriedades de poder alterar o valor da váriavel, mas sem poder ser redeclarada e sofrer hoisting

E o 'const' que como o 'let' não sofre hoisting e não pode ser redeclarada, porém também não pode ter seu valor alterado

Qual a diferença entre for, forEach e for.. of ?

for, forEach e for.. of

As construções for fornecem acesso ao índice do array, não ao elemento real.


              for (let i = 0; i < arr.length; ++i) {
                console.log(arr[i]);
              }
              
              for (let i in arr) {
                console.log(arr[i]);
              }

for, forEach e for.. of

Com as outras duas construções, forEach() e for..of, você tem acesso ao próprio elemento do array. Com forEach() você pode acessar o índice do array em i, com for..of você não pode.


              for (let i = 0; i < arr.length; ++i) {
                console.log(arr[i]);
              }
              
              for (let i in arr) {
                console.log(arr[i]);
              }

for, forEach e for.. of

O que torna as coisas ainda mais confusas é que as construções em loop tratam ['a',, 'c'] diferentemente de ['a', undefined, 'c']. Abaixo está um exemplo de como as 4 construções de looping lidam com ['a',, 'c'] e seu elemento vazio. for..in e forEach pulam o elemento vazio, for e for..of não:


              // Mostra "a, undefined, c"
              for (let i = 0; i < arr.length; ++i) {
                console.log(arr[i]);
              }
              
              // Mostra "a, c"
              arr.forEach(v => console.log(v));
              
              // Mostra "a, c"
              for (let i in arr) {
                console.log(arr[i]);
              }
              
              // Mostra "a, undefined, c"
              for (const v of arr) {
                console.log(v);
              }
